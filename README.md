# Next.js Starter Template

This is a starter template that has a host of goodies and is a great starting place for projects
<br/>
<br/>

## Techonologies & Features

- Next.js
- React
- Tailwind
- Framer Motion
- React Slick Slider
- Axios
- Light & Dark mode options set up
- App Context
- Custom SEO Wrapper
- Custom hooks
  <br/>
  <br/>

## To get up and running

```bash
-> yarn
-> yarn dev
# or
-> npm install
-> npm run dev
```
